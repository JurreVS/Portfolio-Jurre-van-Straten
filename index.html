Blender Character Model

Individual Portfolio Project — 3D Modeling

Overview

For this project, I created an original stylized knight character in Blender. I started by researching Blender, topology, edge flow, character modeling and the general modeling pipeline. After that, I created my own 2D reference and used it to build the first blockout of the character.

During development, I worked on the body, helmet, hands, armor and clothing in separate stages. I ran into problems with things like N-gons, modifiers, clothing intersections and the shape of the helmet, but I was able to solve these through research, troubleshooting and iteration. I also changed the character from a T-pose to an A-pose because this worked better with the shoulder armor and future rigging.

The final result is a complete stylized knight model with clean topology, layered clothing and separate armor pieces. This project mainly focused on modeling and preparing the character for later stages.

Stylized Materials, Shading and Rendering

Individual Portfolio Project — Blender Shaders

Overview

For this project, I turned a blank model into a finished stylized character. I started by researching Blender’s Shader Editor, render engines, material nodes and outline methods. After assigning basic colors, I created a custom shader prototype using Color Ramp, Fresnel and ambient occlusion, then adjusted this setup for the different cloth, leather, metal and helmet materials.

During development, I also added a black interior to the helmet, changed the helmet shader to make it look shinier and created an external outline using the inverted hull method. I refined the lighting, materials and outline thickness by testing the character from different angles. The final result is a fully shaded and rendered stylized knight with custom materials, clear material separation and a complete cartoony look.
