Project Night-Liners

Group Project — Mobile

Overview

A school group project targeted at mobile devices. Night-Liners combines match-3 mechanics with a racing simulation: player performance in the match-3 portion influences a simulated race outcome. The concept balances casual puzzle play with competitive progression and theming.

My contribution

I was the secondary engineer, story writer and UI designer. On the engineering side I implemented scene management, UI functionality, and performed extensive bug fixing. I also helped set up and coach the team on using version control (Git). Additionally I authored the game's theme-based narrative and designed interface flows.

Key Features

  • Dynamic, responsive UI designed for mobile
  • Theme-driven storytelling integrated with gameplay
  • Supporting engineering work: scene & flow management, bug fixing, and team Git support
Itch link: itch.io/project-nightliners

Brushes and Breakthrough

Group Project

Overview

An older school project that follows a failed art student on a journey to regain his passion. The scene begins in a therapist's office and transitions into the artist's mind. Gameplay is a first-person 3D platformer with wall running and sliding, where players navigate obstacles and memories while hearing the therapist's narration.

My contribution

I was the gameplay programmer and voiced the therapist. I implemented advanced movement mechanics (wall running, speed management), dash pickups, checkpoint and respawn systems, and map reset functionality. I also created and implemented all the UI and it's functionality.

Key Features

  • Advanced movement mechanics: wall running, sliding, dashes
  • Intuitive, functional UI across gameplay systems
  • Voice acting (therapist) with synced subtitles to support narrative
Itch link: itch.io/brushes-and-breakthrough
